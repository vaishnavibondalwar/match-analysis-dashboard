Project Title

IPL Match Analysis Dashboard in Excel

Project Objective

The objective of this project is to develop an interactive and dynamic dashboard in Microsoft Excel for analyzing IPL match data and team performance. The dashboard helps users understand match statistics through visual reports and charts, enabling quick insights into winning patterns, toss decisions, stadium performance, and player achievements.

Key Objectives:
Analyze team performance based on batting first and fielding first.
Track toss decision impact on winning percentage.
Identify top venues based on match statistics and total runs scored.
Highlight “Man of the Match” performances.
Provide interactive filtering using slicers for better data exploration.
Present cricket match insights in a visually appealing and user-friendly format.
